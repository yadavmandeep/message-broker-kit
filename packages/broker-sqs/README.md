# @universal-broker/sqs

AWS SQS adapter using @aws-sdk for managed cloud queues.

## Installation

```bash
npm install @universal-broker/sqs
```

## Usage

```typescript
import { SQSBroker } from '@universal-broker/sqs';
import { EnterpriseBrokerWrapper } from '@universal-broker/core';

// Initialize and use with Core wrapper for full features
```

## Features
- Native AWS SQS support
- Seamless integration with @universal-broker/core
- Full TypeScript support
